谢谢你购买我们的模板，下面是安装说明，本模板适用于Magento 1.6.x, 1.7.x 版本。

快捷安装(安装方法1)

1. 从官方网站下载Magento 安装文件并上传到网站空间。
2. 把本模板文件夹 with_samples 下的全部文件拷贝到Magento目录。
3. 用phpMySql或者其他工具导入示例产品数据库(with_sample/database/database.sql 。
4. 用浏览器访问你的网站，按照屏幕提示进行安装。
5. 如需使用其他颜色模板，在 Admin -> System -> Configuration -> Design tab -> Themes->Skin(Images/CSS) 中填入其他颜色模板名称，如 'se105_red', 就是蓝色的模板。
6. 刷新一下cache，重新索引一下 index。
7. 完成。

纯净安装(只安装模板，没有示例产品数据)(安装方法2)

# 文件拷贝

把template_se105下的文件全部拷贝到Magento安装目录

# 把Magento的cache暂时关闭:

在 Admin -> System -> Manage Cache 去暂时关闭cache功能，等模板安装测试成功再打开cache.

# 设置 WISIWYG 编辑器选项:

去 Admin -> System -> Configuration > Content managment 然后设置 WISIWYG 编辑器至 "disabled by default mode".

  • Popular Products是客户浏览最多的产品列表，这个是动态自动生成，可以去后台 CMS->pages->home -> Design -> Layout Update XML field 去设置该块显示产品个数。

  • Bestsellers 是卖的最好的产品列表，这个是动态自动生成。可以去后台 CMS->pages->home -> Design -> Layout Update XML field 去设置该块显示产品个数。

#关于news

本模板已经集成了news 插件,在登陆Magento后台后可以看到News的菜单项，具体使用和问题可以参考官方的文档：News by CommerceLab - User Guide.pdf，该文档已经包括在文件夹里面。

#关于产品页面 tabs

本模板已经集成了easytab 插件,插件地址：http://www.magentocommerce.com/magento-connect/magento-easytabs.html， tab插件的选项如下：

General:

  • Enable easytabs: Yes
  • Enable Description tab :Yes
  • Enable additional info tab: Yes
  • Enable upsell products tab: Yes
  • Enable related products tab :No
  • Enable tags products tab: Yes

Custom Tabs

不用修改，用默认值

Additional CMS Tabs

用默认值，不用修改
