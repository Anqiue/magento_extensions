A)(i) Uploading and installing the theme - top

1. Upload the folders found in the folder named "Theme Upload" to your magento folder on the server. Make sure Magento Cache is disabled.

2. Once done, go to admin » system » design. Click on "ADD DESIGN CHANGE" and select "buyexpress" for the custom design and click save.

For the admin configuration, log out and log back in and go to system » configuration.. You will find the configuration options on top. See picture below:

Change the homepage layout to 1 column. Go to admin » cms » pages and edit "Home page". Click the "design" tab on the left and change the layout to "1column". See image below:

A)(ii) Adding a Featured Attribute - top

  1. Go to admin » catalog » attributes » manage attributes
  2. Click Add new attribute
  3. Fill the text boxes like below:
    • Attribute code: featured
    • Scope : Store View
    • Catalog Input Type for Store Owner: YES/NO
    • Apply To: All product types
    • Use In Layered Navigation: No
    • Use In Search Results Layered Navigation: No
    • Visible on Product View Page on Front-end: Yes
    • Used in Product Listing: Yes
    • Used for Sorting in Product Listing: No
  4. Save the attribute and Go to catalog->attributes->manage attributes sets
  5. Click on Default and drag the featured attribute from the "Unassigned Attributes" under General. See picture below.
  6. Save Attribute set and add a new product. You should see a dropdown with YES/NO. Selecting "YES" will show the product as featured.

B) Theme Configuration - top

The theme's configuration are done from the "BuyExpress Theme Options" from the admin panel..

  1. Go to admin » system » configuration and you should see all the options.

B)(ii) Logo & Favicon Configuration - top

Go to admin » system » configuration and click the First tab and add your content. The first 4 options consists of Favicon., logo, alt text for logo and the Welcome Message. Just upload and add your content. Don't forget to hit the "Save Config" button (top right button).

B)(iii) Social Icon Links - top

Add your social links in fields. If you leave any of the input blank, the corresponding icon will not show up.
