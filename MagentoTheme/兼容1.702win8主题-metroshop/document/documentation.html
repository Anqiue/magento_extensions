Introduction 1

In this document, you will find detailed instructions for installing and using Metrostore Theme. Since the instructions are specific to the theme, they do not cover every Magento function. For help with using Magento, please see the Using Magento section for additional resources Metrostore Theme only supports Magento Community Edition , which can be downloaded from http://www.magentocommerce.com/download The theme does not support Magento Enterprise Edition. Before downloading Magento, make sure you meet the system requirements and test the compatibility of your server. Also, make sure you are using PHP 5.2 or newer. Most of the theme's features are not supported by older versions.

For help with installing Magento, please refer to the following links:
http://www.magentocommerce.com/system-requirements http://www.magentocommerce.com/knowledge-base/entry/how-do-i-know-if-my-server-is-compatible-with-magento http://www.magentocommerce.com/wiki/1_-_installation_and_configuration/magento_installation_guide

Before you begin the Store Theme installation process, we recommend backing up your Magento installation and database.

Smoothing light color design, reveals the passion and attract of luxurious. This Metrostore Magento template is excellent for any type of store website.

Usage Guide 3
To upload Theme to Magento:
  1. Unzip the downloaded folder.
  2. Using an FTP client, copy the app , js and skin folders to the Magento software's root directory.

If you are working with a Mac, please make sure you are merging the new folders with Magento and not replacing the Magento directory.

Metrostore

Now login to the backend, go to System > Configuration , click on Design tab

If you are already using any theme, then please change package name to default.

Note: Make sure to clean cache if it is enabled. To check it go to System > Cache Management . Click Flush Magento Cache.

Homepage Setting Up the Page Layout
To set up the layout of the homepage:
  1. Go to CMS > Pages and select Home page from the list of pages.
  2. Select Enabled in the Status drop-down menu.
  3. Click the Save Page button.
  4. Next step, select Design from the left-hand menu. The following page will appear:
  5. Select 1 column from the Layout drop-down menu.
  6. In the Layout Update XML box, either remove the existing content or add the following block type around the existing content:
  7. Click the Save Page button
Adding Content
