Pinterest Integration

Version 1.0

Pinterest Integration is an extension for Magento. This one allows to integrate Pinterest "Pin It" Button Widget into your store. In a few minutes you can easily install "Pin It" button on each product page of your store. As result description and image of your products are translated into social network Pinterest. This is a great advertisement tool to promote your store and find new customers.

Features

  • Compatible with Canonical Urls
  • Possibility to automatically add price to Product Description
  • Easy-to-use interface
  • Multistore support
  • Easy 2-minutes installation
  • Meets Magento programming practices for most versions and customizations compatibility

Installation

Clear the store cache under var/cache and all cookies for your store domain. This step eliminates almost all potential problems. It's necessary since Magento uses cache heavily.

Backup Your Data

Backup your store database and web directory.

Download and Extract

Download and unzip extension contents on your computer and navigate inside the extracted folder.

Upload Files

If you use a different from default theme - be sure to rename app/design/frontend/default/default and skin/frontend/default/default folders to your store's values. Using your FTP client upload files app, js and skin directory to your store root. Depending on the module some directories mentioned may not exist in the package.

To install "Pin It" button at the product page of your store you should just turn on option "Integrate into Product Page" in the Configuration section of the extension. To manually install this widget you need to open the file:

app/design/frontend/[yourpackage]/[your_theme]/template/catalog/product/view.phtml

and put this code into the place you need: 

<?php echo $this->getChildHtml('mppinterest_direct'); ?>

Configuration

To set up Pinterest Integration go to the System > Configuration > Pinterest Integration page at the backend.

You can enable to display widget at the product page near the Short Description block. Also you can define type of button:

  • Horizontal
  • Vertical
  • Without counter

Troubleshooting

After the extension installation the store gives an error, or blank page, or suggests to start Magento installation procedure.
Change the owner of the extracted extension files to the web server user and set 775 permissions on them. Clear the store cache and try again.
There is no MagPleasure extensions under my configuration section, or having the extension tab clicked I get a blank page, or Access Denied error.
Clear the store cache, browser cookies, logout and login again.
I've set up everything correctly, inserted the HTML code but there is nothing on that page.
Clear the store cache, clear your browser cache and domain cookies and refresh the page.
My configuration changes do not appear on the store.
Clear the store cache, clear your browser cache and domain cookies and refresh the page.
