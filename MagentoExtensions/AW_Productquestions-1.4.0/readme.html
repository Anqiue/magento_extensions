Product Questions

Version 1.4.0

Stay in touch with your visitors and customers with the help of the Product Questions module. It adds a Questions page to each product. Visitors ask questions about products and store administrator answers them.

Then the questions and their answers appear on the Questions page enriching your site content and providing user-demanded text contribution to your items. There is an ability to just reply to the question by email without publishing it.

Besides all benefits the extension gives one more comfortable way to reach store representative and get necessary information. No need to look for a contact form, choosing department, copy-pasting the product title etc. - just click a link on the product page and see what others had asked or type in your question.

Features

  • Email notifications to administrator and customer
  • Easy to use administration backend
  • Private/Public statuses for the questions
  • Templates configuration
  • Multistore support
  • Spam protection
  • SEO-friendly URLs
  • RSS feed
  • Helpfullness rating
  • Configurable question sorting
  • An option to get subscribed (with aheadWorks Advanced Newsletter extension integration)
  • Show last X questions option
  • Auto-responder sending notification to customer
  • Option to parse URLs into links when admin replies
  • A variety of objects available inside the email
  • Easy copy-and-run installation
  • Meets Magento programming practices for most versions and customizations compatibility
  • 100% open source

Installation

Clear the store cache under var/cache and all cookies for your store domain. Disable compilation for Magento 1.4+. This step eliminates almost all potential problems. It's necessary since Magento uses cache heavily.

Backup Your Data

Backup your store database and web directory.

Download and Extract

Download and unzip extension contents on your computer and navigate inside the extracted folder.

Upload Files

Step 1

Navigate inside step_1 directory. If you use a different from default theme - be sure to rename step_1/app/design/frontend/default/default and step_1/skin/frontend/default/default folders to your store's values. Using your FTP client upload content of step_1 directory to your store root.

Step 2

Navigate inside step_2 directory and upload its content to your store root. Some extensions do not have files in step_2 directory, so just ignore this step.

New Page Setup

Product Questions extension will appear in a new page (like Reviews). If you need to show "questions/answers/ask a question from" on your product page (e.g. in tabs), please refer to the next section.

Open file: /your_store_dir/app/design/frontend/default/your_skin_name/template/catalog/product/view.phtml (or, if using Magento version 1.4+, /your_store_dir/app/design/frontend/base/default/template/catalog/product/view.phtml) where your_store_dir is folder, containing your installed store and your_skin_name is name of your current skin. If you use standard magento skin, this should be default.

Find string: 

<?php echo $this->getReviewsSummaryHtml($_product, false, true)?>

and insert the following code right after it: 

<div class="productquestions-summary"><?php echo Mage::helper('productquestions')->getSummaryHtml()?></div>

This will insert "Be first to ask about this product" link just under "Review product" link. If there will be 1+ questions the link will be "Ask a question"

You can also place the summary block by inserting the following line in the appropriate place of the XML product page layout file (if it fit your page design, of course): 

<block type="productquestions/summary" name="productquestions_summary" />

Product Page Setup

You can insert block with product questions directly in your product page via editing your_store_dir/app/design/frontend/default/your_skin_name/layout/catalog.xml (or, in Magento v.1.4+, your_store_dir/app/design/frontend/base/default/layout/catalog.xml)

Place inside the catalog_product_view -> content node 

<catalog_product_view ...>

...
    <reference name="content">
...
<!-- place somewhere around here -->
...
    </reference>

the following lines: 

<block type="productquestions/questions" name="productquestions" >

    <block type="productquestions/sorter" name="productquestions_sorter" />
    <block type="page/html_pager" name="productquestions_pager" />

    <block type="productquestions/form" name="productquestions_form" />
</block>

This will show product questions and "Ask a question" form. To show only questions block, remove the 

    <block type="productquestions/form" name="productquestions_form" />

line from the block above.

To show only "Ask a question" form, insert the 

<block type="productquestions/form" name="productquestions_form" />

line in the appropriate place.


WARNING!!! in Product Questions versions below 1.3 there was other block names: 

<block type="page/html_pager" name="productquestions_list.toolbar" />

<block type="productquestions/productquestions" name="productquestions" template="productquestions/productquestions.phtml" />

It is strongly recommended to remove this old-style lines since it does not implement new Product Questions features and will not be supported in future. Currently there is just a workaround aimed on preventing the product page crash.

Also, remove from the your_store_dir/app/design/frontend/default/your_skin_name/template/catalog/product/view.phtml file the following lines: 

<?php echo $this->getChildHtml('productquestions') ?>

and 

<?php echo $this->getChildHtml('productquestions_form') ?>

if they present there.

Configuration

To configure the extension, navigate to System > Configuration > aheadWorks Extensions > Product Questions.

It is obligatory to save config before using the extension.

Here you can define different options and look how does it fit your store.

Don't forget to fill "Send Emails To" with a correct e-mail address as this is where you would receive email notifications.

Clear Magento cache.

RSS feed

The RSS feed option allows customers to be informed when new question about the product appear. For each product a separate feed is available.

There are RSS Feed group in configuration page of the admin section that allows to set RSS feed title, image, and quantity of questions shown on the feed (ordered by date questions were asked). The RSS feed title can be customized to obtain more details by including the following macros fields:

  • {product_name} - The Product name,
  • {store_name} - The Store name
  • {store_url} - the Store URL,
  • {website_name} - the Website URL

Show last X questions

The Show last X questions option allows you to limit the number of questions displayed on the Product Details page. This option is not applicable for the Product Questions page, and if it is defined, the pager toolbar will not be displayed.

Troubleshooting

After the extension installation the store gives an error, or blank page, or suggests to start Magento installation procedure.
Change the owner of the extracted extension files to the web server user and set 775 permissions on them. Clear the store cache and try again.
There is no aheadWorks extensions under my configuration section, or having the extension tab clicked I get a blank page, or Access Denied error.
Clear the store cache, browser cookies, logout and login again.
I've set up everything correctly, inserted the HTML code but there is nothing on that page.
Clear the store cache, clear your browser cache and domain cookies and refresh the page.
My configuration changes do not appear on the store.
Clear the store cache, clear your browser cache and domain cookies and refresh the page.
